# 机房运维助手

面向学校机房网络运维场景的 AI 助手，当前包含 1 个核心技能：华为交换机远程管理。支持通过 Telnet 远程连接华为 S5700 系列交换机，执行 display 查看命令、分析端口/VLAN/系统状态，自动识别异常并给出运维建议。内置华为 VRP V5.x 命令速查手册。

> **提醒：** 本插件辅助日常运维操作，所有配置变更建议先在测试环境验证后再应用到生产网络。

## 目标角色

- **机房运维人员** — 远程管理交换机、快速查看设备状态、排查网络故障
- **网络管理员** — 批量巡检、VLAN 管理、端口分析

## 快速命令

| 命令 | 说明 |
|------|------|
| `/交换机远程管理` | 远程登录交换机执行运维命令，需提供 IP、端口和密码 |

## 技能

| 技能 | 说明 |
|------|------|
| 交换机远程管理 | 通过 Telnet 连接华为 VRP 交换机，支持分页处理、多命令批量执行，自动分析端口错误/流量异常/资源告警等，附完整 VRP 命令速查 |

## 后续规划

- 服务器巡检（系统资源、服务状态、日志分析）
- 网络故障诊断（分层排查连通性/DNS/DHCP）
- 自动巡检报告生成
- 设备台账管理
