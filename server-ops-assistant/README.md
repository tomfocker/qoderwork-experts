# 个人服务器运维助手

面向个人服务器环境的运维助手插件，覆盖 6 大核心场景：健康巡检、Docker 管理、安全加固、日志分析、备份恢复、资产台账。通过 SSH 远程连接多台服务器（VPS/VM/NAS/路由器/Mac），内置基于真实入侵事件的安全检查经验，支持通过 Agently Mail 推送告警到 QQ 邮箱。适用于 Tailscale 组网的家庭实验室环境，重度 Docker 用户友好。

> **免责声明：** 本插件辅助服务器运维工作，不替代专业安全审计。所有操作结果应由运维人员确认后再执行，特别是涉及防火墙修改、账户变更、数据删除等高风险操作。

## 适用角色

- **个人服务器管理员** — 管理多台 VPS/VM/NAS 等设备的日常运维
- **家庭实验室爱好者** — Docker 容器管理、Tailscale 组网维护
- **安全意识用户** — 基于真实入侵经验的安全审计和加固

## 快速命令

| 命令 | 说明 |
|------|------|
| `/服务器健康巡检` | 批量检查所有设备的 CPU/内存/磁盘/网络/Docker 状态 |
| `/Docker服务管理` | 容器状态检查、启停操作、镜像清理、compose 部署 |
| `/安全加固与防护` | SSH 配置审计、防火墙检查、可疑进程扫描、fail2ban 状态 |
| `/日志分析与告警` | 系统日志分析、暴力破解检测、容器异常检测 |
| `/备份与恢复` | restic 快照管理、容器配置导出、备份验证 |
| `/服务器资产台账` | 查看/更新设备清单、追踪状态变更 |

## 技能详情

| 技能 | 说明 |
|------|------|
| 服务器健康巡检 | SSH 批量连接活跃设备，执行系统资源检查，标注超阈值项（CPU>2x核心/内存>85%/磁盘>85%），生成带状态图标的报告 |
| Docker服务管理 | 容器全生命周期管理，内置排障决策树（日志→资源→重启），重点监控 moviepilot-v2 等已知不稳定容器 |
| 安全加固与防护 | 基于 python3-libstud/escsvc/tempadmin 等真实入侵经验的安全检查清单，涵盖 SSH 加固、防火墙审计、后门检测、SUID 扫描 |
| 日志分析与告警 | 四级告警体系（P0-P3），自动识别暴力破解、异常登录时段、容器重启循环、可疑外连等模式 |
| 备份与恢复 | restic 快照管理 + Docker 配置导出 + 数据库备份，保留策略为 7日/4周/12月，执行前强制 dry-run |
| 服务器资产台账 | 设备清单维护（不存储明文密码），自动追踪变更记录，支持 SSH 在线状态检测 |

## 通知方式

所有技能均支持通过 Agently Mail CLI 将结果推送到 QQ 邮箱。无外部连接器依赖，使用纯 SSH 命令行操作。

> 所有技能在没有外部连接器的情况下均可正常使用，Agently Mail 为可选增强。
